# 집값AI 전략 트랙 (기획/사업)

## 운영 원칙
- 개발 트랙과 분리하여 주 2회 전략 리뷰
- 모든 전략은 수치 KPI와 실험 가설로 관리
- 의사결정은 "가설 → 실험 → 결과" 로그로 기록

## 핵심 KPI
1. 랜딩 방문→계산 완료율
2. 계산 완료→결제 전환율
3. 환불률
4. 재방문율(D7)
5. CAC/회수기간

## GTM 가설 (초기)
- SEO 킬러 키워드(취득세/DSR 계산기)로 유입 확보
- 무료 계산 결과 일부 공개 + 유료 상세 리포트
- 양도세/DSR 불확실성은 "예상"으로 신뢰형 카피

## 우선 실험 4개
1) 가격 실험: 4,900 vs 6,900
2) 결제 시점 실험: 계산 후 즉시 vs 비교표 후 결제
3) CTA 카피: "세금 계산 시작" vs "30초 분석 받기"
4) 리드 획득: 비회원 결과 미리보기 깊이 A/B

## 주간 회의 아젠다
- 이번주 실험 결과
- 실패 지표/원인
- 다음주 가설 2개 선정
